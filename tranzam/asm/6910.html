6910: Screen Address One Pixel Below
Used by the routines at 6900 and 69BC.
Input
HL Current position
Output
HL Address for new position
Calculates the new address for writing a sprite pixel, in an downwards direction.

6911 ADD A,$20
6913 LD L,A
6914 LD A,H H=H + 00 (but include the carry from above).

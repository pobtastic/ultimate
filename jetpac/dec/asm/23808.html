23808: Jetman Frame
Relates to which frame to display.
JETMAN_FRAME 23808 DEFB 215
Value Walking Flying Direction Bits
$00     None 00000000
$01   X Right 00000001
$41   X Left 01000001
$82 X   Right 10000010
$C2 X   Left 11000010
"None" is used to not show Jetman at all i.e. at the start of a game and while "inside the rocket".
